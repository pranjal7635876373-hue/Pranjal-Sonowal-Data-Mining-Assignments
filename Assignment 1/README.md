# MNIST Digit Classifier with TensorFlow Keras

This notebook demonstrates how to build, train, and evaluate a simple Artificial Neural Network (ANN) using TensorFlow Keras to classify handwritten digits from the MNIST dataset.

## Project Structure

The notebook is structured into the following main sections:

1.  **Load and Prepare Data**: Downloads and preprocesses the MNIST dataset.
2.  **Build Neural Network**: Defines a sequential model with dense layers and dropout.
3.  **Compile Model**: Configures the model with an optimizer, loss function, and metrics.
4.  **Train Model**: Trains the neural network on the training data.
5.  **Evaluate Model**: Assesses the model's performance on the test set.
6.  **Confusion Matrix**: Visualizes the model's classification performance using a confusion matrix.
7.  **Classification Report**: Provides detailed per-class precision, recall, and F1-scores.

## Evaluation Results

After training for 5 epochs, the model achieved the following performance on the test set:

*   **Test Accuracy**: 97.92%
